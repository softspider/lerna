# cra-ts-uilib-storybook-lerna

Minimalistic React app in TypeScript based on UI lib with Storybook.
And all of this in Lerna monorepo.

## Installation

```sh
npm run bootstrap
```

## Running Storybook

```sh
npm run story
```

## Running application in dev mode

```sh
npm run start
```

## Build application

```sh
npm run build:app
```
